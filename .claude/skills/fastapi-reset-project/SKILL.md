# FastAPI 项目重置 Skill

## 功能
清空所有学习文档（原始文档和翻译文档），重置学习日志到初始状态，保留 Skills 和项目结构。让新学习者可以从头开始使用该项目。

## 使用方法
在对话中说 "重置项目" 或 "reset project"

## 执行流程

### 1. 清空原始文档
删除 `docs/原始文档/用户指南/` 和 `docs/原始文档/高级指南/` 目录下的所有 .md 文件

### 2. 清空翻译文档
删除 `docs/中文文档/用户指南/` 和 `docs/中文文档/高级指南/` 目录下的所有 .md 文件

### 3. 重置学习日志
将 `docs/学习日志.md` 重置为初始状态：
```markdown
# FastAPI 学习日志

## 学习统计

- **开始日期**: {当前日期}
- **累计学习时间**: 0 小时 0 分钟
- **已完成章节**: 0 / 65 章
- **完成进度**: 0.0%
- **当前进度**: 准备开始

## 学习记录

### {当前日期}

暂无学习记录

## 学习笔记

在此记录学习过程中的重要心得和体会。
```

### 4. 保留文件
**不删除以下内容**:
- `.claude/skills/` - 所有 Skills
- `docs/中文文档/README.md` - 章节目录
- `README.md` - 项目说明
- `CLAUDE.md` - Claude Code 指导文件

### 5. Git 操作提示
重置完成后，提示用户：
1. 检查删除的文件列表
2. 确认无误后提交更改
3. 推送到远程仓库（如果需要）

## 注意事项
- 此操作不可逆，建议在执行前确认
- 会删除所有学习进度和翻译内容
- 不会影响 Skills 配置和项目结构
