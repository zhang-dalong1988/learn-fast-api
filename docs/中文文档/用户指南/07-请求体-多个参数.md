# 请求体 - 多个参数

## 1. 混合路径、查询和请求体参数

当然, 你可以自由地混合 `Path`, `Query` 和请求体参数声明, FastAPI 会知道该做什么。

你也可以通过将默认值设置为 `None` 来声明请求体参数为可选的:

```python
# 导入必要的类
from fastapi import FastAPI
from fastapi import Path, Query
from pydantic import BaseModel

# 创建应用实例
app = FastAPI()


# 定义 Item 数据模型
class Item(BaseModel):
    name: str
    description: str | None = None
    price: float
    tax: float | None = None


# 定义 PUT 路径操作, 包含路径参数、查询参数和请求体
@app.put("/items/{item_id}")
async def update_item(
    *,
    # item_id 是路径参数, 使用 Path 添加验证
    item_id: int = Path(..., title="The ID of the item to get", ge=0, le=1000),
    # q 是查询参数, 使用别名
    q: str | None = Query(None, alias="item-query"),
    # item 是请求体, 可选
    item: Item | None = None,
):
    # 创建包含 item_id 的结果
    results = {"item_id": item_id}
    # 如果提供了 q, 添加到结果中
    if q:
        results.update({"q": q})
    # 如果提供了 item, 添加到结果中
    if item:
        results.update({"item": item})
    # 返回结果
    return results
```

## 2. 多个请求体参数

在前面的示例中, 路径操作期望一个包含 Item 模型属性的 JSON 请求体, 像这样:

```json
{
    "name": "Foo",
    "description": "The pretender",
    "price": 42.0,
    "tax": 3.2
}
```

但你也可以声明多个请求体参数, 例如 `item` 和 `user`:

```python
# 导入必要的类
from fastapi import FastAPI
from fastapi import Path
from pydantic import BaseModel

# 创建应用实例
app = FastAPI()


# 定义 Item 数据模型
class Item(BaseModel):
    name: str
    description: str | None = None
    price: float
    tax: float | None = None


# 定义 User 数据模型
class User(BaseModel):
    username: str
    full_name: str | None = None


# 定义 PUT 路径操作, 包含多个请求体参数
@app.put("/items/{item_id}")
async def update_item(
    *,
    # item_id 是路径参数
    item_id: int = Path(..., title="The ID of the item to get", ge=0, le=1000),
    # item 是请求体
    item: Item,
    # user 是另一个请求体
    user: User,
):
    # 返回包含所有参数的结果
    results = {"item_id": item_id, "item": item, "user": user}
    return results
```

在这种情况下, FastAPI 会期望这样的请求体:

```json
{
    "item": {
        "name": "Foo",
        "description": "The pretender",
        "price": 42.0,
        "tax": 3.2
    },
    "user": {
        "username": "dave"
    }
}
```

## 3. 请求体中的单个值

就像你可以使用 `Query` 和 `Path` 声明额外的查询参数和路径参数一样, 你可以使用 `Body` 声明额外的请求体参数:

```python
# 导入必要的类
from fastapi import FastAPI, Body, Path
from pydantic import BaseModel

# 创建应用实例
app = FastAPI()


# 定义 Item 数据模型
class Item(BaseModel):
    name: str
    description: str | None = None
    price: float
    tax: float | None = None


# 定义 User 数据模型
class User(BaseModel):
    username: str
    full_name: str | None = None


# 定义 PUT 路径操作, 包含多个请求体参数和单个值
@app.put("/items/{item_id}")
async def update_item(
    *,
    # item_id 是路径参数
    item_id: int = Path(..., title="The ID of the item to get", ge=0, le=1000),
    # item 是请求体
    item: Item,
    # user 是另一个请求体
    user: User,
    # importance 是请求体中的单个值, 必填且大于 0
    importance: int = Body(..., gt=0),
    # q 是请求体中的可选单个值
    q: str | None = Body(None),
):
    # 创建结果字典
    results = {"item_id": item_id, "item": item, "user": user, "importance": importance}
    # 如果提供了 q, 添加到结果中
    if q:
        results.update({"q": q})
    # 返回结果
    return results
```

在这种情况下, FastAPI 会期望这样的请求体:

```json
{
    "item": {
        "name": "Foo",
        "description": "The pretender",
        "price": 42.0,
        "tax": 3.2
    },
    "user": {
        "username": "dave"
    },
    "importance": 5
}
```

## 4. 嵌入单个请求体参数

如果你想嵌入单个请求体参数, 可以使用 `Body` 并设置 `embed` 参数:

```python
# 导入必要的类
from fastapi import FastAPI, Body, Path
from pydantic import BaseModel

# 创建应用实例
app = FastAPI()


# 定义 Item 数据模型
class Item(BaseModel):
    name: str
    description: str | None = None
    price: float
    tax: float | None = None


# 定义 PUT 路径操作, 嵌入单个请求体参数
@app.put("/items/{item_id}")
async def update_item(
    *,
    # item_id 是路径参数
    item_id: int = Path(..., title="The ID of the item to get", ge=0, le=1000),
    # item 是请求体, 使用 embed=True 嵌入
    item: Item = Body(..., embed=True),
):
    # 返回结果
    results = {"item_id": item_id, "item": item}
    return results
```

不再是:

```json
{
    "name": "Foo",
    "price": 42.0
}
```

FastAPI 会期望:

```json
{
    "item": {
        "name": "Foo",
        "price": 42.0
    }
}
```

## 5. 总结

你可以:

* 声明多个请求体参数 (`item`, `user` 等)
* 使用 `Body` 在请求体中声明额外的单个值
* 使用 `Body` 并设置 `embed=True` 来嵌入单个请求体参数

以及更多!
