所有代码块都包含详细的中文注释，解释代码的意图和上下文，帮助理解 FastAPI 的工作原理。

## 翻译约定

- 使用中文自然表达，避免直译
- 使用半角标点符号 (,.?!:;)
- 在中文与英文/数字/符号之间、以及在半角符号前添加空格
- 保留专有名词的英文原文（如 FastAPI, Python, Pydantic 等）
- 代码注释解释"为什么"而不是仅仅描述"做了什么"
