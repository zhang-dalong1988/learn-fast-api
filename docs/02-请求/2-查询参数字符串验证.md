# 查询参数和字符串验证

## 1. 额外验证

FastAPI 允许你为参数声明额外的信息和验证.让我们以这个应用为例:

```python
# Python 3.10+
from fastapi import FastAPI

app = FastAPI()

@app.get("/items/")
async def read_items(q: str | None = None):  # q 是可选的字符串查询参数
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

```python
# Python 3.9+
from typing import Union
from fastapi import FastAPI

app = FastAPI()

@app.get("/items/")
async def read_items(q: Union[str, None] = None):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

查询参数 `q` 的类型是 `str | None`, 这意味着它可以是 `str` 类型, 也可以是 `None`, 实际上, 它的默认值是 `None`, 所以 FastAPI 知道它不是必需的.

**注意**: FastAPI 知道 `q` 的值不是必需的, 因为有默认值 `= None`.使用 `str | None` 会让你的编辑器提供更好的支持并检测错误.

## 2. 导入 Query 和 Annotated

要实现验证, 首先导入:

- `Query` 从 `fastapi`
- `Annotated` 从 `typing`

```python
# Python 3.10+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[str | None, Query(max_length=50)] = None  # 使用 Query 进行验证
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

```python
# Python 3.9+
from typing import Annotated, Union
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[Union[str, None], Query(max_length=50)] = None
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

**提示**: 尽可能使用 `Annotated` 版本.

### 2.1 使用 Query 作为默认值(旧方法)

FastAPI 0.95.0 之前的版本需要你使用 `Query` 作为函数参数的默认值, 而不是放在 `Annotated` 中:

```python
# Python 3.9+
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: str | None = Query(default=None, max_length=50)  # 旧方法: Query 作为默认值
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}}]
    if q:
        results.update({"q": q})
    return results
```

**提示**: 对于新代码, 尽可能使用上面解释的 `Annotated`.

## 3. Annotated 的优势

使用 `Annotated` 是推荐的, 它有多个优势:

1. **直观**: 函数参数的默认值是实际的默认值, 这在 Python 中更直观
2. **可重用**: 你可以在没有 FastAPI 的其他地方调用同一个函数, 它会按预期工作
3. **编辑器支持**: 如果有必需参数 (没有默认值), 你的编辑器会报错, Python 也会抱怨
4. **多重注解**: `Annotated` 可以有多个元数据注解, 你甚至可以在其他工具 (如 Typer) 中使用同一个函数

## 4. 添加更多验证

你也可以添加 `min_length` 参数:

```python
# Python 3.10+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[str | None, Query(min_length=3, max_length=50)] = None  # 最小和最大长度
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

```python

@app.get("/items/")
async def read_items(
```

## 5. 添加正则表达式

你可以定义参数应该匹配的正则表达式模式:

```python
# Python 3.10+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[
        str | None,
        Query(
            min_length=3,
            max_length=50,
            pattern="^fixedquery$"  # 正则表达式模式
        )
    ] = None,
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

这个特定的正则表达式模式检查接收到的参数值:

- `^`: 以后面的字符开始, 前面没有字符
- `fixedquery`: 具有精确值 `fixedquery`
- `$`: 就在那里结束, `fixedquery` 后面没有更多字符

**注意**: 在 Pydantic v2 和 FastAPI 0.100.0 之前, 参数叫做 `regex` 而不是 `pattern`, 但现在已弃用.

## 6. 默认值

当然, 你可以使用 `None` 以外的默认值.假设你想声明查询参数 `q` 的最小长度为 3, 默认值为 `"fixedquery"`:

```python
# Python 3.9+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[str, Query(min_length=3)] = "fixedquery"  # 默认值为 "fixedquery"
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

**注意**: 具有任何类型的默认值(包括 `None`) 都会使参数成为可选的(不是必需的).

## 7. 必需参数

当我们不需要声明更多验证或元数据时, 我们可以通过不声明默认值来使查询参数 `q` 成为必需的:

```python
# Python 3.9+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[str, Query(min_length=3)]  # 必需参数, 没有默认值
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

### 7.1 必需但可以为 None

你可以声明参数可以接受 `None`, 但它仍然是必需的.这将强制客户端发送一个值, 即使该值是 `None`.

```python
# Python 3.10+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[str | None, Query(min_length=3)]  # 必需参数, 但可以是 None
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

## 8. 查询参数列表 / 多个值

当你使用 `Query` 明确定义查询参数时, 你也可以声明它接收一个值列表, 或者换句话说, 接收多个值.

```python
# Python 3.10+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[list[str] | None, Query()] = None  # 接收多个值的列表
):
    query_items = {"q": q}
    return query_items
```

```python
# Python 3.9+
from typing import Annotated, Union
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[Union[list[str], None], Query()] = None
):
    query_items = {"q": q}
    return query_items
```

然后, 对于这样的 URL:

```
http://localhost:8000/items/?q=foo&q=bar
```

你将在路径操作函数中, 在函数参数 `q` 中接收多个 `q` 查询参数的值(`foo` 和 `bar`)的 Python 列表.

所以, 对该 URL 的响应将是:

```json
{
  "q": ["foo", "bar"]
}
```

**提示**: 要声明类型为 `list` 的查询参数, 如上例所示, 你需要明确使用 `Query`, 否则它将被解释为请求体.

### 8.1 带默认值的列表

如果没有提供值, 你也可以定义默认值列表:

```python
# Python 3.9+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[list[str], Query()] = ["foo", "bar"]  # 默认值列表
):
    query_items = {"q": q}
    return query_items
```

如果你访问 `http://localhost:8000/items/`, `q` 的默认值将是 `["foo", "bar"]`.

### 8.2 直接使用 list

你也可以直接使用 `list` 而不是 `list[str]`:

```python
# Python 3.9+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[list, Query()] = []  # 直接使用 list
):
    query_items = {"q": q}
    return query_items
```

**注意**: 在这种情况下, FastAPI 不会检查列表的内容.例如, `list[int]` 会检查(并记录)列表内容是整数.但单独的 `list` 不会.

## 9. 声明更多元数据

你可以添加关于参数的更多信息.这些信息将包含在生成的 OpenAPI 中, 并被文档用户界面和外部工具使用.

```python
# Python 3.10+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[
        str | None,
        Query(
            title="Query string",  # 参数标题
            description="Query string for the items to search in the database that have a good match",  # 参数描述
            min_length=3,
            max_length=50,
            pattern="^fixedquery$",
            deprecated=True,  # 标记为已弃用
        ),
    ] = None,
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

### 9.1 别名参数

假设你想要参数名是 `item-query`.但 `item-query` 不是有效的 Python 变量名.最接近的有效名称是 `item_query`.但你仍然希望它在 URL 中是 `item-query`...

你可以这样声明一个 `alias`:

```python
# Python 3.9+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[
        str | None,
        Query(
            alias="item-query",  # URL 中的别名
            title="Query string",
            description="Query string for the items to search in the database that have a good match",
            min_length=3,
            max_length=50,
            pattern="^fixedquery$",
            deprecated=True,
        ),
    ] = None,
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

### 9.2 弃用参数

如果你知道某个参数不再推荐使用, 但出于某种原因仍然需要保留它, 你可以使用 `deprecated=True`:

```python
# Python 3.9+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    q: Annotated[
        str | None,
        Query(
            alias="item-query",
            title="Query string",
            description="Query string for the items to search in the database that have a good match",
            min_length=3,
            max_length=50,
            pattern="^fixedquery$",
            deprecated=True,  # 标记为已弃用
        ),
    ] = None,
):
    results = {"items": [{"item_id": "Foo"}, {"item_id": "Bar"}]}
    if q:
        results.update({"q": q})
    return results
```

### 9.3 从 OpenAPI 中排除参数

要在 OpenAPI 模式中包含一个查询参数, 但在生成的 OpenAPI 模式中排除它(例如, 用于内部参数), 你可以使用 `include_in_schema=False`:

```python
# Python 3.9+
from typing import Annotated
from fastapi import FastAPI, Query

app = FastAPI()

@app.get("/items/")
async def read_items(
    hidden_query: Annotated[str | None, Query(include_in_schema=False)] = None  # 不包含在文档中
):
    if hidden_query:
        return {"hidden_query": hidden_query}
    return {"hidden_query": "Not found"}
```

## 10. 总结

查询参数验证提供了强大的功能:

1. **使用 Query**进行参数验证和元数据声明
2. **Annotated 语法**提供更好的类型安全和可重用性
3. **多种验证规则**:
   - 长度限制(min_length, max_length)
   - 正则表达式匹配(pattern)
   - 默认值
   - 必需/可选参数
4. **高级功能**:
   - 多值参数(list)
   - 参数别名(alias)
   - 标记已弃用(deprecated)
   - 从文档中排除(include_in_schema)
5. **元数据支持**:
   - 标题(title)
   - 描述(description)

这些功能使得 FastAPI 能够构建功能丰富、类型安全的 API.
