# 快速上手

## 1. 安装 FastAPI

```bash
# 安装
pip install "fastapi[standard]"

# 运行开发服务器
fastapi dev main.py
```

## 2. Hello World

```python
from fastapi import FastAPI  # 导入 FastAPI 框架

app = FastAPI()  # 创建应用实例

@app.get("/")  # 路由装饰器: 定义 GET 请求的路径
async def root():  # 异步函数: 提高性能
    return {"message": "Hello World"}  # 返回 JSON 数据
```
